## Features
- Material Design UI
- Easy one-handed operation
- Lightweight
- System-based theme with 3 themes available (Light, Dark & Black)
- Search images with tags
- Image viewer with zoom & pan support
- Download and share image
- DNS over HTTPS enabled by default
